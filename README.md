# 🏎️ Formula 1 Race Simulator

## 🇧🇷 Português

Este projeto consiste em um simulador de corrida de Fórmula 1 desenvolvido em Python.

O usuário escolhe um dos circuitos disponíveis e acompanha uma simulação da corrida, onde diferentes equipes disputam posições de acordo com as características do circuito e do desempenho de cada piloto.

O projeto foi desenvolvido com foco na prática de programação orientada a objetos, modularização do código e interação com o usuário.

---

## Funcionalidades

* Escolha de diferentes circuitos.
* Exibição das informações de cada pista.
* Reprodução de áudio de abertura.
* Simulação da corrida.
* Classificação final dos pilotos.
* Estrutura dividida em múltiplos módulos Python.

---

## Tecnologias utilizadas

* Python
* Pygame
* Programação Orientada a Objetos (POO)

---

## Estrutura do projeto

* `main.py` — arquivo principal da aplicação.
* `f1cars.py` — definição dos carros, pilotos e lógica da corrida.
* `f1racetrax.py` — informações e características dos circuitos.
* `f1audio.mp3` — áudio utilizado na abertura da simulação.

---

## Objetivo

Este projeto foi desenvolvido para praticar:

* Modularização em Python

* Estruturas condicionais e de repetição

The project focuses on object-oriented programming, code modularization and user interaction.
